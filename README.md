# Bad Apple Everywhere

A project that converts any MP4 video into terminal-based ASCII art in real-time, accompanied by MIDI audio. It uses GitHub Actions to automatically compile standalone executables (.exe, Linux binary) and a basic .iso image, allowing the ASCII animation to run across different platforms.



## How to Run

You do not need Python installed to run the pre-compiled versions.



* Navigate to the Actions tab in this repository.
* Click on the latest successful build workflow.
* Scroll down to the Artifacts section.
* Download the desired package:

&#x09;windows-dist: Contains the .exe file for Windows.

&#x09;linux-dist: Contains the Linux binary and the base .iso image.

* Extract and run the executable in your terminal.



*Note: You may need to zoom out (decrease font size) in your terminal for the ASCII art to render correctly without line wrapping.*
